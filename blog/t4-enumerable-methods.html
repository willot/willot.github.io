Today they ask us to speak about one of this three methods that are part of the module Enumerable:
#map #group_by #cycle

But the first question that came in my mind first is what is Enumerable module. After some research I am not sure I totally get it. To make it simple a module is a group a library of methods that you can use across class. But I am sure we will have the occasion to learn more about it.

I had to pick one method to write on so I choose cycle because I am wondering if it can avoid to write loops and simplify the code. Cycle is used to repeat data from an array.
["Tarte", "Pain au chocolat", "Croissant", "Eclair", "Religieuse"].cycle(3) {|food| puts food}

So basically it repeat the data inside the array n times. It actually do the same thing that (n).times. Cycle is a little bit more simple coding.

Another cool thing with cycle is that if you do:
gateaux = ["Tarte", "Pain au chocolat", "Croissant", "Eclair", "Religieuse"]
gateaux.cycle.next => Tarte
If you repeat that 5 time then it iwll be: gateaux.cycle.next => Religieuse
If you do that one more time. gateaux.cycle.next => Tarte
So you get back to the first data in the array.
It is actually pretty cool. Because if you were using: gateaux.each.next it will not go back to Tarte after it reach Religieuse. The reason why is that
gateaux.cycle.object_id == gateaux.cycle.object_id => false
