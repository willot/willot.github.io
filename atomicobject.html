  • Tell me about a time when you were assigned a project that relied on technology unfamiliar to you.

    For our final project at Dev Bootcamp, we created a travel website that used an API. This API came with little to no information on how to use it. We started by finding as much information as we could. From there we realized that the API used a SOAP protocol. We didn't know anything about this protocol. After doing more research, we found a gem called Savon that let Ruby handle the SOAP protocol. Wewere then able to feed and get the data from the API. We organized our team to collect information, experimented, and acknowledged when we were stuck. We went back to doing more research and asked for assistance as needed.
